# Support

## Contact us

**Google Group:** [groups.google.com/g/dpc-api](https://groups.google.com/d/forum/dpc-api) — Ask questions, share feedback, and get updates from the DPC team and other users.

**Email:** [dpcapi@cms.hhs.gov](mailto:dpcapi@cms.hhs.gov) — For issues that require a private response.

::: warning Protect patient data
Never share Protected Health Information (PHI), credentials, or access tokens in support requests. Redact any patient identifiers before sharing API responses or error messages.
:::

## Common questions

**How do I get started?**
Follow the [Quickstart](/quickstart) to make your first API request in the sandbox.

**How do I get production access?**
See [Production](/production/) for requirements and the application process. Note that production onboarding is currently paused.

**What data does DPC provide?**
See [About DPC](/about/) for an overview, or [Data Dictionary](/data-dictionary/) for technical details on FHIR resource types.
